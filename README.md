# spam-detection-email_2
This code will check if the email is spam,important or a promotional email.
The user can load a folder which contains the emails or directly import the email from their account or just upload few txt files.
It uses tkinter to show a gui screen which has various buttons in it.
There is a specific range given in the code from which it will analyses the file and then put it in a category.
The user can save the output in a csv file by clicking on a button in the gui frame.
If too many files,there is also an option to shuffle the files and view the ones which are more import or recent.
